Course Overview

This comprehensive course will teach you the fundamental tools of web development. You'll learn how to create websites using HTML, style them with CSS, and add interactivity with JavaScript.

By the end of this course, you'll be able to build responsive, modern websites from scratch and deploy them to the web. Perfect for absolute beginners with no prior coding experience.

What You'll Learn

  • Structure web pages with HTML5
  • Style websites with CSS3 including Flexbox and Grid
  • Add interactivity with JavaScript (ES6+)
  • Build responsive layouts that work on all devices
  • Work with developer tools and debug code
  • Deploy websites to production using GitHub Pages
  • Understand web accessibility principles

Course Curriculum

  • Introduction to HTML and Web Development
  • HTML Document Structure and Semantic Elements
  • Working with Text, Links and Images
  • Lists, Tables and Media Elements
  • HTML Forms and Input Elements

  • CSS Syntax and Selectors
  • Box Model and Layout Fundamentals
  • Flexbox and CSS Grid Systems
  • Responsive Design and Media Queries
  • CSS Transitions and Animations

  • JavaScript Fundamentals and Syntax
  • DOM Manipulation and Events
  • Working with Forms and Validation
  • Introduction to ES6+ Features

  • Building a Complete Portfolio Website
  • Making it Responsive
  • Deploying to GitHub Pages

Requirements

  • No prior programming experience needed
  • A computer with internet access
  • Any text editor (we'll use VS Code)
  • Modern web browser (Chrome, Firefox, Edge)
